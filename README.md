### Project title
**EduCamp - Android App**

***

### Short description
**EduCamp** is a comprehensive mobile application developed for the **Android platform** to streamline the management and participation in educational camps. It provides users with a seamless experience for browsing available courses, managing registrations, and receiving timely updates and notifications.

***

### Features
This application is designed with the following key features to enhance the user experience:

* **Course Catalog:** Easily browse and view details for all available educational camps and courses.
* **User Registration:** Secure and straightforward process for users to register for specific camps or courses.
* **User Profiles:** Dedicated profiles to manage personal information, view past registrations, and track progress.
* **Notifications & Alerts:** Real-time push notifications for camp schedules, changes, and important announcements.
* **Search Functionality:** Quickly find specific courses or camps using keywords, categories, or filters.
* **Responsive UI:** A clean and intuitive user interface designed with Material Design principles for a smooth experience on all Android devices.

***

### Installation steps
To set up and run the EduCamp Android app locally, follow these steps:

1.  **Prerequisites:** Ensure you have **Android Studio** installed on your system.
2.  **Clone the Repository:** Open your terminal or command prompt and clone the project:
    ```bash
    git clone [https://github.com/dipjoysarker2021/EduCamp-android-app.git](https://github.com/dipjoysarker2021/EduCamp-android-app.git)
    ```
3.  **Open in Android Studio:**
    * Start Android Studio.
    * Select **File** > **Open** and navigate to the cloned `EduCamp-android-app` directory.
    * Android Studio will automatically download the necessary Gradle dependencies. Wait for the project sync to complete.

***

### How to run
After successfully installing the project, you can run the app on an emulator or a physical device:

1.  **Select a Target:** Choose an Android Virtual Device (Emulator) from the dropdown in the toolbar, or connect a physical Android device with USB debugging enabled.
2.  **Build and Run:** Click the **Run** (â–¶) icon in the toolbar, or press `Shift + F10` (or `Control + R` on macOS) to build the app and deploy it to the selected target.
3.  The application should launch on the device/emulator shortly.
